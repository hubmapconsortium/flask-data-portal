We provide /iframe/* routes that are designed for reuse. Normally, we don't worry too much about the target on hrefs, but the default behavior of a link in an iframe is just to open the new page inside the iframe, which is not what we want here.

These examples assume you have a development instance running locally. If configured correctly, links should navigate the whole page, not just the iframe. Remember to set border: none; in the CSS.
